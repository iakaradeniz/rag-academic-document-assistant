\# RAG Academic Document Assistant



A Retrieval-Augmented Generation (RAG) based document assistant designed for academic PDFs.

The system enables context-aware question answering using local LLMs and sentence embeddings.



\## Features

\- PDF document ingestion

\- Page-based chunking with metadata

\- Semantic search using FAISS

\- Local LLM powered question answering

\- Source-aware responses



\## Status

Project initialized. Core ingestion pipeline under development.
